# Personal Portfolio

A clean, responsive personal portfolio / online resume built with plain HTML, CSS, and JavaScript.

## Run
Just open `index.html` in your browser.

## Structure
```
portfolio/
├── index.html      # All page sections
├── style.css       # Styling & responsive layout
├── script.js       # Nav, animations, form
├── assets/
│   └── profile.jpg # Replace with your photo
└── README.md
```

## Customize
Search the source for `TODO:` comments — those mark every place to edit:
- Your name, role and summary (Hero section in `index.html`)
- Bio (About section)
- Skills, Experience, Projects, Education
- Contact info & social links
- Accent color: change `--accent` in `style.css`
- Profile photo: replace `assets/profile.jpg`

## Contact form
The form is a demo. Wire it to a service like Formspree, EmailJS, or your own backend — see `script.js`.
